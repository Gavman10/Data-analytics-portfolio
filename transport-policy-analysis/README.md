# Brisbane Transport Policy Analysis

Did the 50-cent fare policy increase public transport ridership?

## Data

* Public transport ridership data extracted from QLD Data
* Time-series data (pre, during, post COVID)

## Tools Used

* Power BI
* Python (Pandas)

## Key Insights

* Ridership increased ~1.5% after policy introduction
* Growth aligned with post-COVID recovery trends
* Estimated ~10% revenue decline (given that each trip is $3.50)
