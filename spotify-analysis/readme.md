# 🎧 Spotify Data Analytics Project

## Problem

What factors influence the popularity of songs on Spotify?

---

## Data

* Dataset of thousands of Spotify tracks
* Includes features such as energy, danceability, tempo, loudness, and genre
* Built a highly interactive Dashboard that allows searching based on genres and name
* Shows the popularity distribution and top tracks of each artist

---

## Tools

* Power BI
* Python (Pandas)

---

## Key Insights

* Certain genres consistently outperform others in average popularity
* Popular tracks cluster around specific feature ranges rather than extremes
